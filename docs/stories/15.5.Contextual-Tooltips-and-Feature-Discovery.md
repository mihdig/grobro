# Story 15.5: Contextual Tooltips and Feature Discovery

## Status
Approved

## Story
**As a** new user of GroBro,
**I want** First-time tooltips for key features, dismissible and tracked,
**so that** I quickly understand the app's value and get started successfully.

## Acceptance Criteria
1. Contextual Tooltips and Feature Discovery implemented with Smart Greenhouse design
2. Flow is skippable and non-intrusive
3. Progress is tracked and persisted
4. Legal disclaimers displayed clearly
5. User can restart onboarding from Settings
6. Reduces churn in first 7 days

## Dev Notes
- Follow Smart Greenhouse design spec
- Use glassmorphic cards and neon buttons
- Track completion in UserDefaults
- Analytics: track onboarding completion rate
