# Story 12.2: Lock Screen Widgets

## Status
Approved

## Story
**As a** user glancing at my lock screen,
**I want** compact widgets showing next watering time,
**so that** I know at a glance when to water without unlocking.

## Acceptance Criteria
1. Circular widget: countdown to next watering (e.g., "2h")
2. Rectangular widget: "Next: Water Plant A (3h)"
3. Inline widget: "ðŸŒ± Water in 2 hours"
4. Widgets use iOS 16+ lock screen widget API
5. Widgets update via Live Activities for real-time countdown
6. Tapping widget opens app to watering view
7. Widget color adapts to lock screen wallpaper
8. Multiple lock screen widgets can be configured
9. Widgets show critical alerts (e.g., "Overdue: Water now!")
10. Pro users can configure which plant's countdown to show
