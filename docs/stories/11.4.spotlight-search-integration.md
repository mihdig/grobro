# Story 11.4: Spotlight Search Integration

## Status
Approved

## Story
**As a** user searching on my iPhone,
**I want** to find my plants and diary entries via Spotlight,
**so that** I can quickly access specific plants without opening the app.

## Acceptance Criteria
1. All plants indexed in Core Spotlight
2. Searching plant name in Spotlight shows plant result
3. Tapping plant result opens Plant Detail directly
4. Diary events indexed with searchable content (note text)
5. Spotlight results show plant photo thumbnail
6. Spotlight results include metadata: stage, days old, last watered
7. Index updates automatically when plants/events change
8. Deleted plants removed from Spotlight index
9. Privacy: only index non-sensitive content
10. Spotlight results use GroBro icon and branding
