# Story 13.8: Apply Smart Greenhouse Design to PlantCreationView

## Status
Approved

## Story
**As a** new user of GroBro,
**I want** the plant creation flow to have the Smart Greenhouse aesthetic,
**so that** my first interaction with the app feels premium and matches the overall design.

## Acceptance Criteria
1. PlantCreationView uses Smart Greenhouse color palette
2. Form sections use glassmorphic styling
3. Step indicators are visually distinct with design system colors
4. Error states use `.criticalRed` color
5. All form fields have proper accessibility labels
6. Save button uses NeonButton component
7. Works on iOS 17.0+

### Current Issues Found in Review
1. **Line 16**: Standard `Form` styling - no dark background
2. **Line 28**: `.red` for error - should be `.criticalRed`
3. **Lines 20, 34, 63, 71**: `.secondary` - should be `.secondaryText`
4. No glassmorphic styling on sections
5. Standard iOS form appearance

### Design Considerations
- Form is a first-time user touchpoint - must feel premium
- Balance between HIG form conventions and Smart Greenhouse aesthetic
- Step numbers could use `.electricGreen` accent

### Proposed Section Styling
```swift
Section {
    // Content
} header: {
    HStack(spacing: 8) {
        Text("1")
            .font(.caption)
            .fontWeight(.bold)
            .foregroundColor(.deepBackground)
            .frame(width: 20, height: 20)
            .background(Color.electricGreen)
            .clipShape(Circle())

        Text("Plant Details")
            .font(.headline)
            .foregroundColor(.primaryText)
    }
}
.listRowBackground(Color.surfaceDark.opacity(0.5))
```
