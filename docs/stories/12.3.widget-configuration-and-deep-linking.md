# Story 12.3: Widget Configuration and Deep Linking

## Status
Approved

## Story
**As a** user customizing widgets,
**I want** intuitive configuration and seamless navigation from widgets,
**so that** widgets feel like natural extensions of the app.

## Acceptance Criteria
1. Widget configuration UI allows plant selection via picker
2. Configuration shows preview of widget appearance
3. Multiple widget instances can be configured independently
4. Deep links open correct plant detail or action screen
5. Deep links preserve navigation stack (can navigate back)
6. Widget configuration saved per widget instance ID
7. Widgets handle deleted plants gracefully (show fallback state)
8. Configuration UI matches Smart Greenhouse design system
9. Widgets include "Open App" fallback if configured plant unavailable
10. Widget interactions trigger intent donations for Siri suggestions
