# Story 10.2: Search and Bookmarks

## Status
Approved

## Story
**As a** user searching for specific guidance,
**I want** to search all guide content and bookmark useful articles,
**so that** I can quickly find relevant information when I need it.

## Acceptance Criteria
1. Search bar in Guides section searches titles, content, and tags
2. Search results display with highlighted matching text
3. Search supports filters: category, difficulty level
4. Recent searches are saved for quick access
5. Bookmarked guides appear in dedicated "Favorites" section
6. Users can add/remove bookmarks with single tap
7. Bookmarks sync across devices via iCloud (Pro feature)
8. Search is fast (<100ms) using local index
9. Empty search state shows popular guides or suggestions
10. Search analytics track popular queries to inform content creation

## Dev Notes
- Implement using Core Spotlight for iOS integration
- Create inverted index for fast full-text search
- Store bookmarks in SwiftData with iCloud sync
- Use NSPredicate for filtering
