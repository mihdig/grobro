# Story 16.3: Feeding Charts Fox Farm Canna BioBizz

## Status
Approved

## Story
**As a** grower using nutrients,
**I want** Pre-built schedules for Fox Farm Trio, Canna, BioBizz product lines,
**so that** I can feed my plants accurately and track nutrient data alongside environmental and diary events.

## Acceptance Criteria
1. Feeding Charts Fox Farm Canna BioBizz implemented with accurate calculations
2. Data integrates with plant diary timeline
3. Supports major nutrient brands (GHE, Advanced Nutrients, Fox Farm, Canna, BioBizz)
4. Safety warnings displayed prominently
5. Pro features gated appropriately
6. Educational tooltips explain nutrient concepts

## Dev Notes
- Research feeding schedules from official brand websites:
  - Fox Farm: https://foxfarm.com/nutrient-and-fertilizer-schedules
  - Canna: https://www.canna.com/growguides
  - BioBizz: https://www.biobizz.com/feeding-schedules/
- These brands are PRO tier only
- Follow same data structure as 16.2 for consistency
- Store feeding schedules as JSON assets in Resources/
- Apply Smart Greenhouse design system
- Follow source tree: Models in `GroBroDomain/Models/`, JSON in `GroBroFeature/Resources/NutrientSchedules/`, Views in `GroBroFeature/Nutrients/`
- Fox Farm is popular in soil grows
- Canna has separate lines for Terra (soil) and Coco (coco coir)
- BioBizz is organic-focused
